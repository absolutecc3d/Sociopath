# SOCIOPATH

Minimal cyber-style web template.

## Features
- Single HTML file
- Lightweight (~6KB)
- Audio overlay
- Smooth parallax movement
- Easy customization
- No frameworks

## Versions
- V3, similar to V4 but with a glow style
- V4, more simple style with SoundCloud audio support
